Open external links in a new window

  • Set the default in the DCMI wiki to open external links in a new window/tab. According to [1], that means changing a parameter in the LocalSettings.php file to: 
        $wgExternalLinkTarget='_blank';
  • See http://www.mediawiki.org/wiki/Manual:Opening_external_links_in_a_new_window
